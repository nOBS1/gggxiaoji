# 小鸡生蛋：数字合成版设计文档

## 🎯 产品定位
**摸鱼神器** - 像2048一样的数字合成游戏 + 小鸡养成收集元素

## 🎮 核心玩法

### 主游戏：数字合成（2048机制）
- **4×4网格**：经典2048布局
- **操作方式**：
  - PC端：方向键 ↑↓←→
  - 移动端：滑动手势
- **合成规则**：
  - 相同数字碰撞合并（2+2=4, 4+4=8...）
  - 每次操作后随机生成新数字（2或4）
  - 无法移动时游戏结束

### 数字到鸡蛋的映射

| 数字等级 | 鸡蛋类型 | Emoji | 产出奖励 |
|---------|---------|-------|---------|
| 2       | 白蛋 White | 🥚 | 基础货币 |
| 4       | 白蛋 White | 🥚 | 2× 基础货币 |
| 8       | 棕蛋 Brown | 🟤 | 5× 基础货币 |
| 16      | 棕蛋 Brown | 🟤 | 10× 基础货币 |
| 32      | 银蛋 Silver | ⚪ | 20× + 可交易 |
| 64      | 银蛋 Silver | ⚪ | 40× + 可交易 |
| 128     | 金蛋 Gold | 🟡 | 80× + 稀有 |
| 256     | 金蛋 Gold | 🟡 | 150× + 稀有 |
| 512     | 紫蛋 Purple | 🟣 | 300× + 珍稀 |
| 1024    | 紫蛋 Purple | 🟣 | 600× + 珍稀 |
| **2048** | **黑蛋 Black** | ⚫ | **1500× + 终极奖励** |

### 玩法流程

```
开始游戏
   ↓
[4×4空白网格]
   ↓
随机生成2个初始数字(2或4)
   ↓
玩家滑动方向 → 数字移动并合并
   ↓
【合成成功！】
   ↓
根据合成数字等级产出对应鸡蛋 → 计入背包
   ↓
生成新的随机数字(2或4)
   ↓
继续游戏...
   ↓
无法移动 → 游戏结束 → 结算本局收益 → 重新开始
```

4. **产蛋特效**：合成成功后鸡蛋从网格飞向背包

## 🔗 与现有系统集成

### 1. 产蛋系统

```javascript
// 合成成功时调用
function onMergeSuccess(mergedValue) {
  const eggType = getEggTypeByNumber(mergedValue);
  const eggCount = getEggCountByNumber(mergedValue);
  
  // 产蛋到背包
  state.eggs[eggType] += eggCount;
  
  // 触发视觉效果
  showEggReward(eggType, eggCount);
  
  saveGame();
}
```

```javascript
function getEggCountByNumber(num) {
  const baseCount = {
    2: 1, 4: 1,
    8: 1, 16: 2,
    32: 1, 64: 2,
    128: 1, 256: 2,
    512: 1, 1024: 2,
    2048: 5
  };
  
  const levelMultiplier = 1 + (state.upgrades.level * 0.05);
  return Math.floor(baseCount[num] * levelMultiplier);
}
```
