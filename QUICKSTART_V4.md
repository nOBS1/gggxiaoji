## 🎮 玩法说明

### 基础操作
```
PC端: ↑↓←→ 方向键
移动端: 在网格上滑动
```

### 数字合成
```
2 + 2 = 4
4 + 4 = 8
8 + 8 = 16
...
1024 + 1024 = 2048 (终极目标)
```

### 自动产蛋规则

| 合成数字 | 产出鸡蛋 | 数量 | 说明 |
|---------|---------|------|------|
| 2, 4 | 🥚 白蛋 | 1 | 基础 |
| 8, 16 | 🟤 棕蛋 | 1/2 | 常见 |
| 32, 64 | ⚪ 银蛋 | 1/2 | 可交易 |
| 128, 256 | 🟡 金蛋 | 1/2 | 稀有 |
| 512, 1024 | 🟣 紫蛋 | 1/2 | 珍稀 |
| **2048** | **⚫ 黑蛋** | **5** | **终极** |

### 升级加成

**小鸡等级**: 每级+5%产蛋数量  
**幸运加成**: 对金/紫/黑蛋额外+3%/级

例如：
- Lv 10小鸡：产蛋数量×1.5
- Lv 10幸运：稀有蛋×1.3

## 🐛 常见问题

### 1. 游戏没有显示？
**检查控制台**：F12打开开发者工具，查看是否有错误
```javascript
// 应该看到:
✅ 合成游戏初始化完成
```

### 2. 方向键不响应？
- 确保焦点在页面上（点击页面任意处）
- 检查是否有其他元素拦截键盘事件

### 3. 触摸滑动不生效？
- 确认是在"主界面"标签页
- 检查是否在网格区域内滑动

### 4. 产蛋没有计入背包？
- 检查控制台是否有错误
- 验证 `state.eggs` 对象是否正确更新
- 查看"背包"标签页确认

### 5. 游戏卡住无法操作？
- 可能正在动画中，等待300ms
- 刷新页面重新开始
