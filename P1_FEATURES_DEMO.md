## 1. 📝 手续费说明优化

### ✨ 功能描述
创建市场订单时，将简单的一行手续费提示升级为详细的费用明细表。

### 🎯 功能亮点
- **三行明细展示**：
  - 💰 挂单价格 - 白色背景
  - 📉 平台手续费(-5%) - 红色文字突出
  - ✅ 您将收到 - 绿色粗体强调
- **视觉优化**：渐变黄色背景 + hover 动画效果
- **国际化支持**：中英文完整翻译
- **用户提示**：💡 手续费用于维持市场运营

### 📸 使用场景
在创建订单表单中输入价格后，自动显示详细的费用明细。

```
┌─────────────────────────────┐
│ 💰 挂单价格      100 💰     │
│ 📉 平台手续费(5%) -5 💰    │
│ ✅ 您将收到       95 💰     │  ← 绿色粗体
├─────────────────────────────┤
│ 💡 手续费用于维持市场运营   │
└─────────────────────────────┘
```

## 2. ⚠️ 错误提示优化

### ✨ 功能描述
当创建订单或购买订单时，如果库存不足或金币不足，显示详细的错误信息。

### 🎯 功能亮点
- **库存不足详细提示**：
  ```
  库存不足！当前: 2个，需要: 5个
  ```
- **金币不足详细提示**：
  ```
  金币不足！当前: 50💰，需要: 100💰
  ```
- **后端支持**：RPC 函数返回 `current` 和 `required` 字段
- **国际化支持**：中英文动态翻译

### 📸 使用场景
1. 创建订单时库存不足
2. 购买订单时金币不足
3. 任何涉及资源检查的操作

### 🔧 技术实现
- 前端：模板字符串变量替换 `{current}` → 实际数值
- 后端：错误响应中包含 `error.data.current` 和 `error.data.required`

## 3. 📄 订单列表分页

### ✨ 功能描述
市场订单列表底部添加完整的分页导航组件。

### 🎯 功能亮点
- **分页信息显示**：
  - 显示当前范围和总数 (如 "1-20 / 156")
- **页码导航**：
  - 上一页 / 下一页按钮
  - 智能页码显示（当前页 ± 1 + 首尾页）
  - 省略号表示跳过的页码
- **每页数量选择**：
  - 可选择 10 / 20 / 50 条每页
  - 切换后自动重置到第一页
- **交互优化**：
  - 翻页后自动滚动到订单列表顶部
  - 筛选/排序时自动重置到第一页
- **响应式设计**：移动端自动调整布局

### 📸 使用场景
浏览大量市场订单时，通过分页快速定位目标订单。

### 🎨 UI 展示
```
┌───────────────────────────────────────┐
│  1-20 / 156  ← [1] [2] [3] ... [8] → │
│                 每页: [20 ▼]          │
└───────────────────────────────────────┘
```

### 🔧 智能页码算法
- 总页数 ≤ 5：显示所有页码
- 总页数 > 5：显示 `1 ... 当前-1 [当前] 当前+1 ... 最后`
- 示例：共 8 页，当前第 4 页 → `1 ... 3 [4] 5 ... 8`

## 4. 🔄 自动刷新功能

### ✨ 功能描述
在市场标签页中，每 30 秒自动刷新市场订单和统计数据。

### 🎯 功能亮点
- **智能启动/停止**：
  - 切换到市场标签 → 自动启动刷新
  - 离开市场标签 → 自动停止刷新
- **刷新提示**：
  - 右上角浮动提示 "刷新中..."
  - 带有加载动画图标
  - 滚入/滚出动画效果
- **防重复刷新**：
  - 检测当前是否正在加载
  - 避免在加载过程中重复刷新
- **刷新内容**：
  - 市场订单列表
  - 市场统计数据

### 📸 使用场景
用户停留在市场标签页时，无需手动刷新即可看到最新的订单和价格。

### 🎨 UI 展示
```
右上角浮动提示：
┌─────────────┐
│ ⏳ 刷新中... │  ← 动画滑入
└─────────────┘
```

### ⏱️ 时间设置
- **刷新间隔**：30 秒
- **动画时长**：300 毫秒（滑入/滑出）

## 🎮 使用指南

### 体验手续费优化
1. 进入市场标签 → 市场广场
2. 在"创建订单"表单中输入价格
3. 查看自动显示的详细费用明细

### 体验错误提示
1. 尝试创建订单但库存不足
2. 或尝试购买订单但金币不足
3. 查看详细的错误提示信息

### 体验分页功能
1. 在市场广场浏览订单
2. 滚动到底部查看分页组件
3. 点击页码或切换每页数量

### 体验自动刷新
1. 进入市场标签页
2. 停留 30 秒观察右上角刷新提示
3. 切换到其他标签，刷新自动停止
