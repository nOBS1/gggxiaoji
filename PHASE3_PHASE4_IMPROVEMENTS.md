## 📝 技术笔记

### 手续费优化实现细节

1. **HTML 结构** - 使用嵌套 div 创建分层结构
   ```html
   <div class="fee-preview">
     <div class="fee-info">
       <div class="fee-row">...</div>
       <div class="fee-row fee-deduction">...</div>
       <div class="fee-row fee-total">...</div>
     </div>
     <p class="fee-hint">...</p>
   </div>
   ```

2. **CSS 技巧**:
   - 使用 `linear-gradient` 创建渐变背景
   - `transform: translateX(4px)` 实现 hover 滑动效果
   - `rgba(255, 255, 255, 0.7)` 半透明白色创建层次感
   - `box-shadow` 和 `border` 配合增强视觉效果

3. **JavaScript 更新**:
   - 添加 `listingPrice` 元素的更新
   - 保持原有的 `feeAmount` 和 `receiveAmount` 更新
   - 格式化添加 💰 emoji

4. **国际化**:
   - 所有新增文本都支持中英文
   - 保持一致的翻译键命名规范
