## 📊 技术栈详情

### 前端框架
- **构建工具**: Vite 5.4.20
- **JavaScript**: ES6+ (Modules)
- **CSS**: 原生 CSS (无预处理器)
- **HTML**: HTML5

### 开发规范
- 模块化开发（ES6 Modules）
- 命名规范：
  - 变量/函数：camelCase
  - 常量：UPPER_SNAKE_CASE
  - CSS 类：kebab-case
- 注释：中文
- 代码风格：统一缩进、空格

### 项目结构
```
xiaoji-game/

│   └── music/          # 音效文件

│   │   ├── main.css
```
