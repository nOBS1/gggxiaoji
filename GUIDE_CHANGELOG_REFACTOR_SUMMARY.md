## 💻 JavaScript 核心功能

### 1. 打开弹窗
```javascript
function openChangelogModal() {
  changelogModal.classList.add('active');
  document.body.style.overflow = 'hidden';
  localStorage.setItem('lastViewedChangelog', new Date().toISOString());
  localStorage.setItem('changelogVersion', '2.2.0');
}
```

### 2. 版本检查
```javascript
function checkForNewVersion() {
  const lastViewedVersion = localStorage.getItem('changelogVersion');
  const currentVersion = '2.2.0';
  
  if (!lastViewedVersion || lastViewedVersion !== currentVersion) {
    floatingChangelogBtn.classList.add('has-update');
  }
}
```

## 🚀 使用说明

### 查看游戏指南
1. 打开游戏，停留在主界面
2. 向下滚动，小鸡下方即可看到
3. 四个指南卡片清晰展示

### 查看更新日志
1. 点击右下角 📢 "更新日志" 按钮
2. 弹窗显示所有版本历史
3. 点击关闭按钮/背景/ESC键关闭

### 新版本提示
- 首次访问或版本更新时，浮动按钮显示红点
- 点击查看后，红点自动消失
- 记录保存在localStorage中
