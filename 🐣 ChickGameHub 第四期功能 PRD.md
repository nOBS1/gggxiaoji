## 一、功能概述

玩家在第三期已具备稳定的 **产蛋 → 卖蛋 → 获得金币** 的经济循环。
 在第四期中，引入 **股票市场系统**，使金币具备投资属性，从而延伸出第二层经济玩法：

> 🪙 金币 → 📈 股票投资 → 📊 股价波动 → 💰 盈亏收益

核心目标：

- 为玩家提供“投资成长感”与“财富增值体验”；
- 增强金币的用途，避免经济通胀；
- 为未来事件驱动（行业新闻 / 市场情绪）打下机制基础。

### 3.3 股价波动算法（伪代码）

```js
// 每5分钟或玩家访问市场时更新
function updateStockPrice(stock) {
  const base = stock.current_price
  const randomFactor = (Math.random() - 0.5) * 2 * stock.volatility // -volatility ~ +volatility
  const trendFactor = (stock.trend - 1) * 0.02                      // 趋势偏移
  const newPrice = base * (1 + randomFactor + trendFactor)
  stock.current_price = Math.max(1, parseFloat(newPrice.toFixed(2)))
  stock.last_update = new Date()
  return stock
}
```

后期可扩展：

- 行业新闻事件影响波动；
- 玩家买卖量对价格产生“轻微反应”（需求驱动型波动）；
- 特定节日、公告、灾害等“事件卡”干预价格。

------

### 3.4 交易规则

| 项目     | 说明                                        |
| -------- | ------------------------------------------- |
| 买入     | 使用金币按当前股价购买，可设定数量。        |
| 卖出     | 按当前股价出售，获得金币。                  |
| 手续费   | 每次交易收取固定比例（例如 1%）。           |
| 持仓上限 | 可按等级或VIP权限增加（防止囤积）。         |
| 刷新机制 | 每5分钟刷新一次所有股价，前端动态动画展示。 |
| 风险提示 | 股价可跌至1金币（但不会为负）。             |

------

### 3.5 玩家收益与排行榜

- 玩家收益率 = (当前总市值 + 可用金币) / 投入金币 - 1
- 系统每周计算“投资收益榜”（前100名）；
- 可发放“理财达人”称号 / 特殊头像框；
- 后期可开放联盟投资、企业IPO等扩展机制。
