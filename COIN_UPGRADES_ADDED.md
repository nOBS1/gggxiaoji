```

```
名称: 金币加成
描述: 提升卖蛋获得的金币
成本: 80金币 (每级×1.8倍)

## 🔧 技术实现

### 1. 配置文件 (config.js)
```javascript
luckyChance: {
  name: '幸运加成',
  desc: '提升所有稀有蛋掉落率',
  costType: 'coins',  // 使用金币
  baseCost: 50,
  maxLevel: 15,
  baseValue: 5  // 每级+5%
}
```

### 2. 状态管理 (state.js)
```javascript
upgrades: {
  level: 1,
  feed: 0,
  clickPower: 0,
  idleRate: 0,
  luckyChance: 0,    // 新增
  autoSell: 0,       // 新增
  goldBonus: 0       // 新增
}
```

### 3. 游戏逻辑 (gameLogic.js)

#### 幸运加成 - 影响掉落权重
```javascript
if (luckyChance > 0) {
  const bonus = 1 + (luckyChance * 0.05);
  weights.brown *= bonus;
  weights.silver *= bonus;
  weights.gold *= bonus;
  weights.purple *= bonus;
  weights.black *= bonus;
}
```

#### 金币加成 - 影响卖蛋收益
```javascript
const baseCoins = CONFIG.RARITIES[rarity].price * amount;
const goldBonus = state.upgrades.goldBonus;
const bonusMultiplier = 1 + (goldBonus * 0.1);
const coins = Math.floor(baseCoins * bonusMultiplier);
```

#### 自动售卖 - 被动卖蛋
```javascript
const autoSellLevel = state.upgrades.autoSell;
if (autoSellLevel > 0 && state.eggs.white > 0) {
  const sellAmount = Math.min(state.eggs.white, autoSellLevel * 5);
  sellEgg('white', sellAmount);
}
```

### 4. UI显示 (ui.js)
```javascript
// 金币升级特殊样式
if (config.costType === 'coins') {
  card.style.borderColor = 'var(--primary)';
  card.style.background = 'linear-gradient(135deg, #fffbeb 0%, #fef3c7 100%)';
  
  // 显示金币成本
  const have = state.coins;
  const color = have >= cost ? 'green' : 'red';
  costHTML = `<span style="color: ${color};">💰 ${cost}</span>`;
}
```
